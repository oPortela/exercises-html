Common Abbreviations

A sigla HTML represneta a linguagem de marcação de hipertexto, usada em aplicações web.

Em aplicações web utilizamos também o CSS, ele faz com que colocamos o estilo na página deixando ela bonita.

Outra ferramento que usamos é banco de dados que é feito para armazenar as informações que podem ser utilizadas na página, a linguagem padrão de um banco de dados é o SQL, ela á usada para a manipulação dos dados.

E não podemos esquecer do WWW que se refere à rede mundial de computadores.
